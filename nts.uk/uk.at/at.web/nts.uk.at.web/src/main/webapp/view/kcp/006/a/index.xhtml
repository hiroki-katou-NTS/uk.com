Description
DateObject start:The date/time of date. Required.A Moment-ish input, like an ISO8601 string ('YYYY-MM-DD').
textColor:Sets list text's text color. Required, rgb code or color name.
headerBackgroundColor:Sets date's header backgroud color. Required, rgb code or color name.
backgroundColor:Sets date's backgroud color. Required, rgb code or color name.
listText:List text display in the date. Required, array of string.

buttonClick Triggered when the user clicks on each button:
