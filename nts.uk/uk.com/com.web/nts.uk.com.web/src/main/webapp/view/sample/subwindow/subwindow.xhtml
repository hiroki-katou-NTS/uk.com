SubWindow

Overview

A SubWindow is a control which open another page with given URL. Basically it's a dialog with <iframe> inside.






Don't style functions-area on your own
- Use id="functions-area" for function area on top
- Use id="functions-area-bottom" for function area on bottom Close button and Title!
- Hide or show close button is decided by Kiban team so don't try to change it. Please ignore design document if there is any different
- Dialog title will be auto display by Kiban team so don't mind about that too

Methods

Name

Description
modal(url, option?) Open a modal sub-window, which you can't interact with parent window.
Return: ScreenWindow
Agruments:
url: string Relative URL usually start with "/view"
option: object (optional) Dialog option extend from JqueryUI Dialog option
modeless(url, option?) Open a modal sub-window, which you can interact with parent window.
Return: ScreenWindow
Agruments:
url: string Relative URL usually start with "/view"
option: object (optional) Dialog option extend from JqueryUI Dialog option
