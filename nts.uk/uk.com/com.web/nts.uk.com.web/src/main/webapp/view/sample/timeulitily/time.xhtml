Time ulitily

In this module have functions for work with time, date in javascript. Below is main functions of it.

formatYearMonth

Parameter: (seconds: number, formatOption: string)

This method will receive a number as parameter. And convert it into year month formay (yyyy/MM).

formatSeconds

Parameter: (seconds: number, formatOption: string)

Convert input number according to format

parseTime

Parameter: (time: string or number, isMinutes: boolean (optional))

Parse input value into time format (h:mm). Parameters consist input value and is input value minute? Return value is ResultParseTime

"1:23"
nts.uk.time.parseTime(1123.1243);
Result: result.format() ->"11:23"
nts.uk.time.parseTime("11:2");
Result: result.format() ->"11:02"
nts.uk.time.parseTime(90, true);
Result: result.format() ->"1:30"]]>

parseYearMonth

Parameter: (value: string or number)

Parse input value into year month format (yyyy/MM). Return value is ResultParseYearMonth

"1999/02"
nts.uk.time.parseYearMonth("1999/02");
Result: result.format() ->"1999/02"]]>

parseTimeOfTheDay

Parameter: (value: string or number)

Parse input value into time of day format (hh:mm). Return value is ResultParseTimeOfTheDay

 fail
nts.uk.time.parseTimeOfTheDay(1902);
Result: result.format() -> "19:02"
nts.uk.time.parseTimeOfTheDay(9902);
Result: result.format() -> fail
nts.uk.time.parseTimeOfTheDay("19:02");
Result: result.format() -> "19:02"]]>

parseYearMonthDate

Parameter: (value: string or number)

Parse input value into date format (yyyy/MM/dd). Return value is ResultParseYearMonthDate

 "1999/02/11"
nts.uk.time.parseYearMonthDate("1999/02/11");
Result: result.format() -> "1999/02/11"]]>

formatDate

Parameter: (date: Date, format: any)

Format date value according to format inputed.

 "1999-02-11 00:00:00/000"]]>

yearInJapanEmpire

Parameter: value (string or number)

Format year value according to japan calendar format. Method return a object is JapanYearMonth. In JapanYearMonth have properties are empire, year, month which can use method get to get data or can use toString method.

yearmonthInJapanEmpire

Parameter: value (string or number)

Format year value according to japan calendar format. Method return a object is JapanYearMonth.
